Particle Filter

Observation Module: $\Pr[s_t|o_t]$ this is an observation model, it outputs the probability of an estimated belief by the pure observation. In practice, the observation model estimate the belief state and assign each observed entity to the current belief state.
Physics Module: $\Pr[s_t|s_{t-1},\theta]$, $\Theta(s_{t-1})$ the intuitive physics model is a conditional distribution $\Pr[s_{t}|s_{t-1}]$. Evolution of the belief state is given by sampling from this conditional distribution $\Pr[\cdot|s_{t-1}]$
Matching Module: $\Pr[o_t|s_t]$ this is the matching model, it estimate the probability of a belief state matches certain observation. It is noticed that there are several entities not observable: wind, air etc.
Knowledge Module: $E$ this model represents knowledge E. Physics model defined a raw model that can perform base level prediction. In a more abstract level, we can have a set of transition rules like: if $x$ is a pipe and $y$ is a *flowable* object, then $put(y,x,t)\rightarrow\exists t'>t,around(y,x.end)$. How the flowable object go? the flowable object flow through the pipe. (these detailed parameterization can be implemented using diffusion model etc.)
$$s_t^m=\Theta(s_{t-1}^m),w_t^m=w^m_{t-1}\Pr[o_t|s_t^m]$$ In practice, the observation module is implemented using a pre-trained perception module, it takes the raw image (scene-graph) as input, and output an *ObjectSet*. An ObjectSet is a tuple in the following form $\{m_i,s_i\}$, where $m_i$ is the mask of the object and $s_i$ is the probability of existence. The internal state space (belief) is modeled as a distribution over ObjectSets $s_t$ for some frame $t$. specifically the belief model holds a set of $M$ particles $\{s_t^m\}_{m=1}^M$ with normalized weights $\{w_t^m\}_{m=1}^M$ to approximate the belief distribution denotes as $\Pr[s_t,o_{< t}]$. Each step these particles goes throw a stochastic transition model and update weights of each component of the belief state. and the observation model is written as $$\Pr[o_t|o_{< t} ]=\int_{s_t}\Pr[o_t|s_t,o_{< t}]\Pr[s_t|o_{< t}]\mathrm{d}s_t=\sum_{m=1}^Mw^{m}_t\Pr[o_t|s_t^m]$$ The implementation of this code can be seen at.
